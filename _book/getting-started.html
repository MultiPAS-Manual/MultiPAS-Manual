MultiPAS User’s Guide

3 Getting Started

This section outlines the basic steps needed to start sampling with the PAS for the first time and after it has been turned off. It assumes the instrument has been connected according to Chapter 2.

3.1 Installation

  1. Download and install the following third-party software:
    • LabVIEW run time engine 2011 or later (National Instruments)
    • WaveIO (Christian Zeitnitz)
    • OBIS Connection (supplied on USB flash drive, Coherent, Inc.)
  2. Download the MultiPAS-III (3 wavelengths) or MultiPAS-IV executable.
  3. Connect the USB cable to the computer and let Windows Update install the necessary drivers.

3.2 Powering On

  1. Switch the main power on by moving the switch on the rear of the control box to the on position (“1”).
  2. Ensure the laser supplies have illuminated indicators. If necessary, turn their power switches to on.
  3. Open the laser apertures by sliding the knurled slider on top of the laser heads in the PAS case.
  4. Ensure the microphone is plugged in to DC power and it’s green LED is illuminated.
  5. Turn the photodiode power supply on via the switch on the power brick. Turn the photodiode on via the switch on the side of the sensor; the green light should illuminate.
  6. Remove the photodiode’s lens cap if necessary.
  7. Turn the pump on by switching on the “Pump” switch on the front of the control box to begin pulling sample through the instrument.

3.3 Using the Software

  1. Open the “MultiPAS-III.exe” control program, which should start automatically. Note: If COM ports have been changed, or a new computer is used you may have to stop the program and select the appropriate COM ports in the first tab of the UI.
  2. Type your calibration coefficient into the appropriate box in the Calibration Tab to display quantitated absroption data.
  3. Turn the lasers on by pressing All Emit. Alternatively, individual lasers can be turned on and off with the buttons for each laser. You should see three peaks near 1370 Hz in the FFT plots under both the Microphone and Photodide tabs.
  4. To have the instrument operate autonomously, write the full filepath (including the name) into the file name boxes in the first time of the software.
    • Save Sweeps: Saves the automatic frequency sweeps upon completion; format Frequency, Hz (column 1) and Intensity, mV (column 2).
    • Save Raw: Saves the microphone and photodiode raw data; format (for each, by column) Time (%Y-%m-%d %H:%M:%S), 423 nm data (mV), 532 nm data (mV), 785 nm data (mV). It is suggested to always save raw data.
    • Save Absorption: Saves the absorption values caluculated in real time; format (by column) Time (%Y-%m-%d %H:%M:%S), 423 nm data (1/Mm), 532 nm data (1/Mm), 785 nm data (1/Mm). Note: It is suggested to always save raw data, even when saving the calculated absorption.
  5. To automatically collect background spectra, select Auto Filter on the main panel of the UI. Under this setting the control box will automatically cycle between HEPA filtered air (background) and sample, and will perform frequency sweeps in between. You may set the sample time (time between filters) and background time (time for the filter and frequency sweep) to the desired time, but 24 minutes and 6 minutes are, respectively, the default and suggested values.
